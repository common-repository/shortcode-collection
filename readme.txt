Adds buttons to tinyMCE Visual Editor so that no need to remember shortcodes. 

== Description ==

This plugin will add buttons to TinyMCE Visual Editor which when clicked will append shortcode where the cursor is placed in the Editor.
You can customize the settings from Shortcode Collection Dashboard ->Settings.
It reduces your burden of remembering all the shortcodes.
Also if You have More shortcode already active in the blog, then Shortcode collection Plugin comes with a dashboard widget and a
Post Editor Meta box which gives you a list of all the active shortcodes form your blog.
Also there comes settings to disable them if they are no more helpful to you.



**Features**

* Buttons to add shortcodes.
* No need of remembering the shortocdes.
* Customize which shortcode you like to use.
* A Dashboard Widget Listings all the active Shortcodes.
* A Meta Box in the post-page Editor which does the same as Dashboard widget.

== Installation ==

1. Upload `shortcode-collection` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Shortcode Collection Dashboard -> Settings and customize them as you like and enjoy!

== Frequently Asked Questions ==

= What is Pirate Talk? =

It replaces some words of the content placed between [pirate][/pirate] with the slang. So those who know the slang only those can understand what you want to say!

= What is signoff text? = 

Its just the text that appears at the place of the [signoff] shortcode. Also there is a signoff Text content option to be filled so that you can decide what content to appear instead of the shortcode.
